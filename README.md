# 🚀 基于 YOLOv11 的人脸识别实时考勤系统

## 📌 项目简介
本项目是一款基于 **YOLOv11-face** 模型的轻量化人脸识别考勤系统。通过调用本地摄像头进行实时人脸检测，并结合本地人脸数据库（`face_db`）进行身份匹配，最终实现自动考勤记录与 CSV 报表导出。

本项目旨在解决传统考勤系统对复杂库依赖过高的问题，采用了原生 Python 逻辑处理数据持久化，确保了系统在不同环境下的稳定性和兼容性。

## ✨ 核心功能
* **实时人脸检测**：采用最新的 YOLOv11-face 模型，具备极高的检测速度和准确率。
* **自动化考勤**：系统自动扫描 `face_db` 文件夹进行匹配，实现“无感签到”。
* **智能去重**：内置考勤逻辑，确保同一时段内单人不会重复计入。
* **本地化存储**：无需数据库环境，程序退出后自动生成带时间戳的 **CSV 报表**。
* **高兼容性**：去除了对 `pandas` 等重型库的依赖，极大降低了部署门槛。

## 🛠️ 技术栈
* **核心框架**：Python 3.9+
* **深度学习推理**：Ultralytics / YOLOv11
* **图像处理**：OpenCV-Python
* **数据处理**：CSV (Standard Library)
